## 🇷🇺 О проекте DeadloRUS
Всем привет! Представляю вам **DeadloRUS** — проект русской локализации для героев Deadlock. Проект находится на старте и поэтому **огромная** часть реплик не переведена. **Реплики переводятся по мере возможности и свободного времени.**

В рамках этого проекта реплики персонажей поэтапно переводятся и записываются на русском языке, а затем аккуратно обрабатываются через нейросетевые голосовые модели. Моя главная цель — максимально точно сохранить оригинальные тембры, уникальные характеры и изначальную энергию героев, сделав их речь живой и понятной для русскоязычных игроков. Список озвученных персонажей будет постепенно пополняться!

## ⚙️ Установка (Installation)

Установить мод можно двумя способами — через лаунчер или вручную.

### Способ 1: Через Deadlock Mod Manager (Рекомендуется)
1. Скачайте архив `deadlorus_vX.X.X.zip` из раздела **[Releases](../../releases/latest)** (в правой колонке страницы).
2. Откройте Deadlock Mod Manager.
3. Перейдите в раздел **Мои моды** (My Mods).
4. Нажмите **Добавить локальный мод** (Add Local Mod / Import) и выберите скачанный `.zip` архив.
5. Запускайте игру с модами!

### Способ 2: Ручная установка
1. Скачайте архив `deadlorus_vX.X.X.zip` из раздела **Releases**.
2. Распакуйте архив. Внутри вы найдете файл `deadlorus.vpk` (он лежит по пути `game/citadel/addons/`).
3. Переместите файл `deadlorus.vpk` в папку с игрой по пути:  
   `Steam\steamapps\common\Deadlock\game\citadel\addons\`  
   *(Если папки `addons` нет — просто создайте её).*
4. Запустите игру.

**To Valve and original IP holders:**
This is a strictly non-commercial, fan-made localization project created out of love for **Deadlock**. 
* **No Gameplay Advantage:** This mod only replaces client-side audio files (`.vpk` addon) and does not provide any competitive advantage, alter gameplay mechanics, or interact with memory.
* **Repurposing of Assets:** The audio for the characters was manually translated, acted, and recorded by a human, carefully preserving the original intonation and emotional context. A voice model, trained exclusively on the original English voice lines of the respective characters, was then applied strictly as an audio filter to adapt the timbre. The resulting audio is applied only back into the exact same game to create a Russian localization.
